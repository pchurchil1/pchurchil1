# What I’m Building:
## DriveOps — Vehicle Telemetry Platform

A full-stack fleet monitoring and telemetry system designed to demonstrate real production engineering skills.

FastAPI backend with PostgreSQL, SQLAlchemy, Alembic, and pytest
React + TypeScript dashboard for vehicle, ECU, and event monitoring
Redis-backed API gateway with rate limiting and metrics
Dockerized local development environment
AWS deployment plan using ECS Fargate, ECR, RDS, ElastiCache, ALB, CloudFront, Secrets Manager, CloudWatch, Terraform, and GitHub Actions CI/CD

## Outlook Smart Sort

A local-first Outlook VSTO add-in that uses machine learning to organize emails into user folders with human approval.

Built with C#, .NET Framework, Outlook Interop, SQLite, and ML.NET
Supports top-3 folder suggestions, batch approval, fuzzy folder search, undo, and retraining
Designed around real-world constraints: local data privacy, Outlook performance, large inboxes, and user trust
